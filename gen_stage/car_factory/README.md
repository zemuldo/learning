# CarFactory

## Description

Car Factory example steps:

- Create prototype with props:
  - Base Base Prototype (v1, v2, v3)
  - Model (3, V, W, X, Y)
  - Color (Black, Silver, Blue, Red, White, Maroon)
  - Estimated time to finish (Default 6 weeks)

- Implement prototype
  - Apply propulsion version
  - Apply os version
  
- Apply model
  - Apply tire version
  - Apply body version

- Apply color
  - Apply pain
  - Apply tainting

- Apply Test drive
  - Apply test drive.
